# Claude Code Router 快速参考卡片

## 🚀 常用命令

### 服务管理
```bash
ccr start          # 启动路由器
ccr stop           # 停止路由器
ccr restart        # 重启路由器
ccr status         # 查看状态
```

### 使用 Claude Code
```bash
ccr code           # 通过路由器启动 Claude Code
```

### 配置管理
```bash
ccr ui             # 打开 Web 配置界面
ccr model          # 交互式模型选择器
ccr preset         # Preset 管理
```

---

## 📝 配置文件位置

```
~/.claude-code-router/config.json
```

---

## 🔧 最小化配置

```json
{
  "Providers": [
    {
      "name": "deepseek",
      "api_base_url": "https://api.deepseek.com/chat/completions",
      "api_key": "sk-xxx",
      "models": ["deepseek-chat"]
    }
  ],
  "Router": {
    "default": "deepseek,deepseek-chat"
  }
}
```

## 🎯 路由规则速查

| 场景 | 配置项 | 说明 |
|------|--------|------|
| 默认 | `default` | 所有未特别指定的请求 |
| 后台任务 | `background` | 轻量级、非关键任务 |
| 深度思考 | `think` | 复杂推理任务 |
| 长上下文 | `longContext` | 超过令牌阈值的请求 |
| 网络搜索 | `webSearch` | 涉及网络搜索的请求 |
| 图像处理 | `image` | 包含图像的请求 |

## ⚡ 快速开始步骤

1. **安装**
   ```bash
   npm install -g @anthropic-ai/claude-code
   npm install -g @musistudio/claude-code-router
   ```

2. **配置**
   ```bash
   ccr ui  # 或使用文本编辑器创建 config.json
   ```

3. **启动**
   ```bash
   ccr start
   ```

4. **使用**
   ```bash
   ccr code
   ```

## 🔐 安全建议

- ✅ 使用环境变量存储 API 密钥
- ✅ 设置 `APIKEY` 限制访问
- ✅ 将 `HOST` 设置为 `127.0.0.1`（本地）
- ✅ 定期清理日志文件
- ✅ 不要将配置文件提交到 Git

## 📊 日志级别

| 级别 | 用途 |
|------|------|
| fatal | 致命错误，系统无法运行 |
| error | 错误，但系统可继续运行 |
| warn | 警告信息 |
| info | 一般信息 |
| debug | 调试信息（开发用） |
| trace | 详细跟踪（深度调试） |

## 💻 推荐工作流

### 日常开发
```json
"default": "ollama,qwen2.5-coder:latest"  // 本地模型，快速响应
"background": "ollama,qwen2.5-coder:latest"  // 后台任务也用本地
```

### 复杂任务
```bash
/model deepseek,deepseek-reasoner  // 需要推理时切换
```

### 生产环境
```json
"default": "openrouter,claude-sonnet-4"  // 高质量模型
"fallback": ["deepseek,deepseek-chat"]  // 降级方案
```

## 📈 成本优化策略

1. **日常开发**：使用本地 Ollama 模型（免费）
2. **中等难度**：使用 DeepSeek（经济实惠）
3. **复杂任务**：使用 Claude/Gemini Pro（高质量）
4. **批量处理**：使用更便宜的背景模型
